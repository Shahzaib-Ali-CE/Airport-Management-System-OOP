# Airport-Management-System-OOP
A C++ Object-Oriented Programming project that manages flight ticket booking, cancellation, modification, passenger records, and persistent data storage using file handling.

## Technologies Used

- **Programming Language:** C++
- **Programming Paradigm:** Object-Oriented Programming (OOP)
- **IDE:** Dev-C++
- **Concepts:** Composition, Single Inheritance, Multilevel Inheritance, Multiple Inheritance, Function Overloading, Friend Functions
- **File Handling:** Text File Storage
- **Data Validation:** Input Validation
